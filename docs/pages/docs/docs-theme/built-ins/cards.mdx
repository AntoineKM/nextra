# Card Component

A built-in card component of the Docs Theme.

## Example

<Cards>
  <Card title="Card 1" href="#example" />
  <Card title="Card 2" href="#example" />
  <Card title="Card 3" href="#example" />
</Cards>

## Usage

### Default

```mdx
import { Card, Cards } from 'nextra-theme-docs'

<Cards>
  <Card title="Card 1" href="#default" />
  <Card title="Card 2" href="#default" />
  <Card title="Card 3" href="#default" />
</Cards>
```

### Number of Cards per Row

You can use the `num` prop to specify the number of cards per row. The default
is `3` and the maximum is also `3`.

```mdx
import { Card, Cards } from 'nextra-theme-docs'

<Cards num={2}>...</Cards>
```

<Cards num={2}>
  <Card title="Card 1" href="#number-of-cards-per-row" />
  <Card title="Card 2" href="#number-of-cards-per-row" />
  <Card title="Card 3" href="#number-of-cards-per-row" />
</Cards>

### With an Arrow

```mdx
<Cards>
  <Card arrow title="Card" href="#" />
</Cards>
```

### With an Image

```mdx
import Image from 'next/image'

<Cards>
  <Card title="Card with an image" image href="#">

  </Card>
</Cards>
```
